Detailed Description

template<typename T>
class containers::Vector< T >

Vector class Vectors are dynmacially resized, generic containers. This vector implmentation may differ somewhat from a standard implementation in that the underlying data uses a unique pointer, meaning it can only have a single owner at any given time.
